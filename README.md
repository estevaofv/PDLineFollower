# Proportional Derivative Line Follower

## Components:
* Arduino Mega 2560
* Pololu QTR 8A Sensor
* L298N Motor Driver
* 7.4V Lipo Battery
* Magician Chassis


### The Documentation is being updated, check `LineFollower.h` file for information.
